Loris

Loris is an Open Source C++ class library implementing analysis, manipulation, and synthesis of digitized sounds using the Reassigned Bandwidth-Enhanced Additive Sound Model. Loris supports modified resynthesis and manipulations of the model data, such as time- and frequency-scale modification and sound morphing. Loris includes support and wrapper code for building extension modules for various scripting languages (Python, Tcl, Perl). A graphical interface to some of the sound modeling and morphing functionality of Loris is provided by Fossa, a QT-based GUI written by Susanne Lefvert. For more information, please visit the Loris web page.

Loris is developed by Kelly Fitz and Lippold Haken at the CERL Sound Group , with contributions from Paul Christensen .
Loris is is available for download from SourceForge, please visit the Loris project page .
